# 🎬 SHOWZY

A modern Full Stack Movie Ticket Booking Platform built using **React.js** and **Django REST Framework**.

## 🚀 Features

- 🎥 Browse Movies
- 🎟️ Book Tickets
- 💺 Interactive Seat Selection
- 👥 Group Booking
- 🍿 Food Pre-order
- 📱 QR Code Ticket Generation
- 📧 Email Confirmation
- 🔐 JWT Authentication
- 👤 User Dashboard
- 🏢 Theatre Admin Dashboard
- 👑 Super Admin Dashboard
- 📊 Revenue Analytics

## ⚙ Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend

```bash
cd frontend
npm install
npm start
```
